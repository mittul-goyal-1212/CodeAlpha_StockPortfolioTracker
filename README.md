# CodeAlpha_StockPortfolioTracker

A simple stock tracker that calculates total investment value based on manually defined stock prices.

## Features
- Hardcoded dictionary of stock prices
- User inputs stock symbols and quantities
- Validates stock symbols and rejects non-numeric quantities
- Displays total investment value
- Optionally saves results to a .txt or .csv file

## Key Concepts Used
dictionary, input/output, basic arithmetic, file handling (optional)

## How to Run
```bash
 python Stock_Portfolio_Tracker.py
​```
